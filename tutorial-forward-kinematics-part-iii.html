Tutorial: forward kinematics. Part II

Posted by Erick Vieyra on Mon 05 September 2016

In this post we will define the DH matrices needed to create the kinematic model of an ABB IRB120 industrial robot that has 6 degrees of freedom We will use the things we learned from the first and second parts of this tutorial and we will use that to create a Python script that showcases the kinematic model of the robot.

The ABB IRB120 robot

The IRB120 is ABB's smallest industrial robot. It has 6 degrees of freedom, which makes it a versatile robot for its size. We will get the dimentions of the robot from the publicly available CAD drawings from ABB Robotics.

Creating the kinematic model

Now we will define and place the coordinate systems in the robot and get the corresponding DH parameters:

Note: The position of the robot in the CAD was taken as the robot's zero-position.

  1. We will place in the base of the robot the origin [0,0,0] of the first frame.
  2. The frame associated to the joint in the base has its origin in [0,0,0.561] regarding the base frame. The corresponding DH parameters are: \(δ = 0.561\), \(θ = 0\), \(r = 0\),\(α = 0\).
  3. After placing the \(Z\) axis in the next joint, we can see that the axis is rotated 90 deg in comparison th the \(Z\) axis of the previous joint. In DH parameter terms, it has a 90 degree rotation in \(α\). We place the origin in [0,0,0.309]. The DH parameters are: \(δ = 0.309\), \(θ = 0\), \(r = 0\), \(α = 90\).
  4. The next joint has its \(Z\) axis in the same direction as the last one, but it has an offset of 0.810m from the origin of the last frame. Give the orientation of this coordinate system, we will set \(θ\) with an angle of 90 so we can have the current position of the robot as the the zero-position. The DH parameters are \(δ = 0\), \(θ = 90\), \(r = 0.810\), \(α = 0\).
  5. The next frame will be place just 0.21m above the last one, with its \(Z\) axis pointing towards the last two joints of the robot (90 degree rotation). The DH parameters are \(δ = 0\), \(θ = 0\), \(r = 0.21\), \(α = 90\).
  6. A distance of 0.906m separates our frame with the coordinate system of the next segment. This one has no rotation whatsoever. The DH parameters are \(δ = 0.906\), \(θ = 0\), \(r = 0\),\(α = 0\).
  7. Lastly, we will place our coordinate system with a rotation of 90 degrees so that the \(Z\) axis points towards the end effector of the robot. We will specify a small offset for the tool of the robot in the next step. The DH parameters are \(δ = 0\), \(θ = 0\), \(r = 0\) , \(α = 90\).
  8. The last segment is inside the wrist of the robot. We will add a last frame that represents the end effector of the robot. The offset for this frame will be 0.216m, according to the CAD drawing.

Computing the robot's kinematics with Python

The following script defines the needed DH matrices for an IRB120 robot, applies the joint values to said matrices and plots a representation of the robots state.

from mpl_toolkits.mplot3d import axes3d
import matplotlib.pyplot as plt
import numpy as np
from itertools import product, combinations
import math as m

print("test")
# Set plot related things...
fig = plt.figure()
ax = fig.add_subplot(111, projection='3d', aspect='equal')
plotRange = 3
ax.set_xlim(-plotRange*0.3,plotRange*0.7)
ax.set_ylim(-plotRange*0.5,plotRange*0.5)
ax.set_zlim(0,plotRange)

def dhMatrix(d,t,r,a): # delta, theta, r, alpha
    """Gets the Denavit-Hartenberg parameters and
    returns a numpy-based Denavit-Hartenberg matrix"""
    M = np.matrix([
    [m.cos(t) ,-m.sin(t)*m.cos(a) , m.sin(t)*m.sin(a) ,r * m.cos(t)],
    [m.sin(t) , m.cos(t)*m.cos(a) ,-m.cos(t)*m.sin(a) ,r * m.sin(t)],
    [0          , m.sin(a)            , m.cos(a)              ,d           ],
    [0            , 0                           , 0                         ,1           ]])

    return M

def drawFrame(F):
    """Draws a coordinate reference frame from a 4x4 matrix.
    Adds an extra thick line along the Z axis and around the
    origin to simulate the joint in a robot segment"""

    R = F[0:3,0:3] # extract Rot matrix
    T = F[0:3,3]    # extract traslation matrix
    p1 = np.matrix([[0.5],[0],[0]]) # get the 3 other points of the drawn frame
    p2 = np.matrix([[0],[0.5],[0]])
    p3 = np.matrix([[0],[0],[0.5]])
    p1 = R*p1   # Rotate the points
    p2 = R*p2
    p3 = R*p3
    x = T.item(0)   # Get the origin's position of the frame
    y = T.item(1)
    z = T.item(2)
    #draw the line from the origin to each of the points
    ax.plot([x,x+p1.item(0)],[y,y+p1.item(1)],[z,z+p1.item(2)], color="b")
    ax.plot([x,x+p2.item(0)],[y,y+p2.item(1)],[z,z+p2.item(2)], color="k" )
    ax.plot([x,x+p3.item(0)],[y,y+p3.item(1)],[z,z+p3.item(2)], color="r")
    l = 0.1 # draw the extra thicker line that simulates the
    #robot's joint
    ax.plot([x-p3.item(0)*l,x+p3.item(0)*l],
            [y-p3.item(1)*l,y+p3.item(1)*l],
            [z-p3.item(2)*l,z+p3.item(2)*l],
            linewidth=6, color = "k")

def drawTool(F):
    """Draws a very simple body from a 4x4 matrix to represent the
    end effector of a robot."""

    R = F[0:3,0:3] # extract Rot matrix
    T = F[0:3,3]    # extract traslation matrix
    p1 = np.matrix([[0.5],[0],[0]]) # get the 3 other points of the drawn frame
    p2 = np.matrix([[0],[0.5],[0]])
    p3 = np.matrix([[0],[0],[0.5]])
    p1 = R*p1   # Rotate the points
    p2 = R*p2
    p3 = R*p3
    x = T.item(0)   # Get the origin's position of the frame
    y = T.item(1)
    z = T.item(2)

    l = 0.1 # draw the extra thicker line that simulates the
    #robot's joint
    ax.plot([x-p2.item(0)*l,x+p2.item(0)*l],
            [y-p2.item(1)*l,y+p2.item(1)*l],
            [z-p2.item(2)*l,z+p2.item(2)*l],
            linewidth=6, color = "r")


def drawSegment(F1,F2):
    """Draws a thick segment between two Frames
    given as arguments."""
    T1 = F1[0:3,3]  # traslation matrix 1
    T2 = F2[0:3,3]  # traslation matrix 2

    ax.plot([T1.item(0),T2.item(0)],
            [T1.item(1),T2.item(1)],
            [T1.item(2),T2.item(2)],
            linewidth=3, color = "k")

sA = 3 * (3.1415/6) # straight angle 90 degrees

# MODIFY THE THETA PARAMETERS HERE TO SEE THE CHANGES!
To  = dhMatrix(0        ,0    ,0      ,0)
T01 = dhMatrix(0.561    ,0    ,0      ,0)   # first joint to second
T12 = dhMatrix(0.309    ,0    ,0      ,sA)  # second to third joint
T23 = dhMatrix(0        ,sA   ,0.810  ,0)
T34 = dhMatrix(0        ,0    ,0.21   ,sA)
T45 = dhMatrix(0.906    ,0    ,0      ,0)
T56 = dhMatrix(0        ,0    ,0      ,sA)

Ttool = np.matrix('1 0 0 0 ; 0 1 0 0.216 ; 0 0 1 0 ; 0 0 0 1')


T02 = T01 * T12
T03 = T02 * T23
T04 = T03 * T34
T05 = T04 * T45
T06 = T05 * T56
Tt  = T06 * Ttool


drawFrame(To)
drawSegment(To,T01)
drawFrame(T01)
drawSegment(T01,T02)
drawFrame(T02)
drawSegment(T02,T03)
drawFrame(T03)
drawSegment(T03,T04)
drawFrame(T04)
drawSegment(T04,T05)
drawFrame(T05)
drawSegment(T05,T06)
drawFrame(T06)

drawTool (Tt)

plt.show()

print Tt
print ("End effector position submatrix [xyz]:")
print (Tt[0:3,3:4])
print ("End effector orientation submatrix")
print (Tt[0:3,0:3])
